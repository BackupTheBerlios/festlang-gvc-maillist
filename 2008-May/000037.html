Hi all,

I've been thinking on gvc word recognition system (well, in sphinx) and I
have a proposal:

What's about getting phonetic transcription of each word automatically
instead of reading it from a text file?

That is, when a window gets focus, we get its items by means of
accessibility description. So, maybe, we can get phonetic transcription of
each item if we would can find a way to translate it into phonemes. Then,
sphinx could compare with these phonemes (that have been generated on the
fly). The main goal of this method is the flexibility: a big list with all
possible words isn't necessary now, and any accessible application could be
use from gvc, regardless of the words that it uses.

I don't know how difficult is this method in english, but, in Spanish, for
instance, there is a direct translation from letters  (or groups of letters)
to phonemes.

Moreover, this is a common feature in mobile phones, where you can say the
name of your contacts. I don't believe it store a big list with all possible
names...
